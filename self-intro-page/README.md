# self-intro-page — 自己紹介ページをAIと作ろう

AIからの**11の質問に答えるだけ**で、あなたの自己紹介ページ（Webページ）ができあがるツールです。
できたページは無料でインターネットに公開して、URLをみんなに共有できます。

プログラミングの知識はいりません。ぜんぶ日本語のおしゃべりで進みます。

## できあがるもの

スマホでもパソコンでもきれいに見える、1枚の自己紹介ページです。

- 名前・ふりがな・部署・拠点と、プロフィール画像（写真がなければ頭文字のきれいな丸アイコン）
- SNSリンクと発信ジャンル（SNSをやっている人だけ。なければ出ません）
- これまでの歩み（略歴のタイムライン）
- わたしの強み3つ（カード）
- すきなこと（タグ）
- 大事にしている言葉
- さいごに、ひとこと

色や雰囲気はあなたの好みに合わせて変わります。

## はじめかた（Claude Code派・社内推奨）

1. **GitHubからダウンロード**: 配布リポジトリ https://github.com/levela-inc/withai-skills を開き（要ログイン・招待制）、緑の**「Code」ボタン→「Download ZIP」**でzipを保存します
2. **zipを展開**: Macはzipを**ダブルクリック**、Windowsは**右クリック→「すべて展開」**
3. **フォルダを置く**: 展開してできた `self-intro-page` フォルダを、自分の作業フォルダ（いつもClaude Codeを使う場所）に置きます
4. **Claude Codeを開く**: その作業フォルダでClaude Codeを開いて「**自己紹介ページを作って**」と送ります
5. あとは質問に1つずつ答えるだけ。プレビューの確認も、修正も、公開も、Claude Codeにおまかせできます

### Cursor派

1. Cursorのチャット欄を開きます
2. `PROMPT.md` を開いて、中身を**全部コピー**してチャットに貼り付けて送信します
3. あとは質問に1つずつ答えるだけ。ファイルの保存や公開のコマンドはCursorがやってくれます

### ChatGPT派

1. `PROMPT.md` を開いて、中身を**全部コピー**してChatGPTに貼り付けて送信します
2. 質問に1つずつ答えると、ページのHTML（コードブロック）が出てきます
3. 保存のしかた（コピーして `index.html` という名前で保存）もChatGPTがそのまま案内してくれます
   - ChatGPTはあなたのパソコンのファイルを直接さわれないので、保存だけ自分の手で行います

## プロフィール画像を使いたいとき（任意）

ChatGPTなどで作ったイラストや、お気に入りの写真を、ページのいちばん上のアイコンに使えます。

- 質問の途中でAIが「画像を self-intro フォルダに入れましたか？」と聞いてくれるので、そのタイミングで画像ファイルを入れればOK
- 画像がなくても大丈夫。名前の頭文字を使ったきれいな丸アイコンになります
- **注意**: ページを公開すると、フォルダに入れた画像もいっしょに公開されます

## インターネットに公開する（Surge）

できたページは **Surge**（サージ）というサービスで無料公開できます。

- Claude Code・Cursorなら「**Surgeで公開して**」と頼むだけ
- 自分でやる場合は、ターミナル（文字でパソコンに指示を出す画面）で次のコマンドを打ちます

```
npx surge self-intro 好きな名前.surge.sh
```

例: `npx surge self-intro mataken-intro.surge.sh`

- 初回だけメールアドレスとパスワードの登録を求められます（無料）
- 公開されると `https://好きな名前.surge.sh` というURLになり、誰でも見られるようになります
- `self-intro` フォルダごと公開されるので、中に入れたプロフィール画像も一緒に公開されます

## 気をつけること

- **公開したページは世界中の誰でも見られます。** 電話番号・住所・会社の内部情報（社外秘）は載せないでください
- フォルダに入れた画像も公開されます。公開してよい画像だけ入れてください
- 迷ったら「これは載せて大丈夫？」とAIに聞いてみてください

## 困ったら

- エラーの文が出たら、**その文をそのままAIに貼り付けて**「どうしたらいい？」と聞くのがいちばん早いです
- それでも解決しなければ、Discordで気軽に質問してください
